# restaurant

## Backend

Made with python 3.8 + flask

To run it simply execute `python api.py`

## Frontend

Made with Vue 3 + Vuex + Router + Axios + Bootstrap


### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
